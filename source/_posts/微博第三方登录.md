## 微博第三方登录的坑

网页接入微博第三方登录，在这遇到好多坑，特此记录一下，终于出坑了，开心

在最开始使用微博第三方登录是用的纯前端来做的，使用微博端的jssdk来进行授权登录流程, 这是登录的代码，这基本上就是按照官方文档里写的，

```js
WB2.login(function () {
    // 验证是否登入成功
    if (WB2.checkLogin()) {
        // api 入口 
        WB2.anyWhere(function (W) {
            // 调用 account/get_uid 接口，获取用户信息                        
            W.parseCMD('/account/get_uid.json', function (oResult, bStatus) {
                console.log(oResult, bStatus, "oResult,bStatus")
                if (bStatus) {
                    console.log(1)
                    // 本地验证 uid 是否存在，如果存在则自动登入绑定账户，不存在则不做任何操作
                    islogin(oResult.uid)
                }
            }, {}, {
                method: 'get',
                cache_time: 30
            });
        });
    }
});
```

<!--more-->

上边然而并无卵用，于是各种资料查询，又研究官方文档，按照它给的demo各种尝试， 但是授权并不能成功，折腾好久无用，发现WB2.login(callback)，这个功能应该是废弃了，于是只能自己进行授权获取用户id了，本来想着前端直接搞定就行了，然而并不行

```JS
var wburl = window.location.href
window.location.href =
    `https://api.weibo.com/oauth2/authorize?client_id=XXXXXXXX&redirect_uri=${wburl}&response_type=code`
//                     
```

通过这边获取授权code 请求https://api.weibo.com/oauth2/access_token?client_id=XXXXXXXXXX&client_secret=XXXXXXXX&grant_type=authorization_code&redirect_uri=XXXXXXXXXX&code=XXXXXXXXXXX   在前端进行请求这个页面的时候又跨域问题 然而 ajax的jsonp只能进行get请求，走到这发现光靠前端是无法实现了，没办法授权的流程只能交给后端来做了，但是总体的原理就是这样的  

现在的做法如下  部分代码

```js
 /* 判断微博是否授权成功 */
/* 获取授权code*/
var wbcode = window.location.href.split('code=')[1];
console.log(wbcode)
/*判断code存在*/
if (wbcode) {
    /* 调用后端接口获取用户id**/
    $.ajax({
        url: `/wbreturn.aspx?number=${localStorage.checknumer}&code=${wbcode}`,
        dataType: 'json',
        type: 'GET',
        success: function (res) {
            console.log(res)
            if (res.msg == 'success') {
                //  成功获取后登录
                islogin(res.openid)
            }
        },
        error: function () {
            $.prompting('请求失败')
        }
    })
}
/* 点击微博登录调用的方法**/
function wblogin() {
    /*获取后端设置的checknumber*/
    $.ajax({
        url: '/WebJson.aspx?book=Getchecknumber',
        dataType: 'json',
        type: 'GET',
        success: function (res) {
            console.log(res)
            if (res.msg == 'success') {
                var wburl = window.location.href
                /*获取微博的code吗**/
                window.location.href =
                    `https://api.weibo.com/oauth2/authorize?client_id=xxxxx&redirect_uri=${wburl}&response_type=code`
				
                localStorage.checknumer = res.checknumer
            }

        }
    })
```
